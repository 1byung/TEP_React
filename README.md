# Tennessee Eastman Process (TEP) 대시보드

완전히 새로운 디자인의 실시간 공정 모니터링 대시보드입니다.

## 🎨 디자인 특징

- **Modern Minimal Design**: 깔끔하고 현대적인 카드 기반 레이아웃
- **Blue/Purple Gradient Theme**: 부드러운 그라데이션 배경
- **Smooth Animations**: Framer Motion을 활용한 부드러운 애니메이션
- **Real-time Updates**: 1초마다 업데이트되는 실시간 데이터

## 📊 주요 기능

### 1. KPI 카드 (상단)
- **평균 온도**: 온도 센서 평균값
- **평균 압력**: 압력 센서 평균값
- **평균 유량**: 유량 센서 평균값
- **시스템 정상률**: 전체 센서 상태 퍼센트

### 2. 실시간 센서 데이터 차트 (중앙 좌측)
- 4개의 선택된 센서 실시간 모니터링
- 30초 슬라이딩 윈도우
- 색상별 센서 구분 (Blue, Pink, Green, Orange)
- 부드러운 라인 차트

### 3. 센서 상태 히트맵 (중앙 우측)
- 40개 센서의 실시간 상태를 색상으로 표시
- 색상 코딩:
  - 🟢 초록: 낮음 (0-40)
  - 🔵 파랑: 중간 (40-70)
  - 🟡 노랑: 높음 (70-100)
  - 🔴 빨강: 경고
- 호버 시 확대 효과

### 4. 이상치 탐지 게이지 (하단 좌측)
- 원형 프로그레스 바로 시스템 정상률 표시
- 색상 변화:
  - 초록: 90% 이상
  - 노랑: 70-90%
  - 빨강: 70% 미만
- 정상/주의/경고 센서 개수 표시

### 5. 최근 이벤트 로그 (하단 우측)
- 실시간 이벤트 로그
- 이벤트 타입별 색상 구분:
  - 파랑: 정보
  - 노랑: 경고
  - 초록: 정상
- 자동 스크롤

## 💾 데이터 시뮬레이션

### TEP 센서 (52개)
- `XMEAS_1` ~ `XMEAS_52`
- 센서 타입:
  - 온도 (1-13)
  - 압력 (14-26)
  - 유량 (27-39)
  - 조성 (40-52)

### 실시간 업데이트
- 1초마다 모든 센서 값 업데이트
- 랜덤하게 ±5 범위 내에서 변동
- 5% 확률로 이벤트 로그 생성
- 차트 데이터는 30초 슬라이딩 윈도우로 유지

## 🎯 특징

### 반응형 디자인
- 모바일, 태블릿, 데스크탑 모두 지원
- Grid 시스템으로 자동 레이아웃 조정

### 부드러운 애니메이션
- 카드 로딩 시 페이드인 효과
- 히트맵 호버 시 확대 효과
- 이벤트 로그 추가 시 슬라이드 애니메이션
- 게이지 업데이트 시 스무스한 전환

### 직관적인 색상 코딩
- 각 KPI마다 다른 색상 테마
- 상태별 명확한 색상 구분
- 그라데이션 효과로 시각적 깊이감
